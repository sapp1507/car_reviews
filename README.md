# car_reviews
## Тестовое задание "Отзывы об автомобилях"

Описание проекта
----------------
Проект создан в рамках тестового задания

Проект разварачивается в Docker контейнерах:

backend - приложение api, 
postgresql - база данных
nginx - Web сервер

Стек технологий
----------

* python 3.10.7
* Django 4.1.1
* Rest API
* PostgreSQL
* Nginx
* Docker
